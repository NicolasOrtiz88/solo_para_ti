# Proyecto Genesis 🌸✨

Un rincón mágico interactivo lleno de detalles lindos, animaciones fluidas y efectos visuales románticos para alegrar el día.

---

## 🐟 Destacado: Pecera Virtual Hello Kitty (`Espero_te_guste.html`)
El proyecto principal y más especial de este repositorio es una **Pecera Interactiva Kawaii** inspirada en **Hello Kitty**.

### 🌟 Características Clave:
* **Hello Kitty en Vectores (SVG):** Diseñada completamente en código vectorial con su listón rojo clásico, orejitas, whiskies y una tierna nariz amarilla.
* **Expresiones Dinámicas:** Al tocar al pez o alimentarlo, guiña el ojo y hace una carita de beso muy linda.
* **Física de Alimentación:** Al presionar "Dar de Comer 🍲", caen copos de comida realistas en el agua. El pez detecta la comida, nada a buscarla y se pone feliz al comerla.
* **Burbujas y Luces Dinámicas:** Efectos de burbujas ascendentes, algas marinas con balanceo natural y rayos de sol animados que cruzan el agua.
* **Chat Kawaii en 3 Idiomas (ES, FR, PT):** Un chat interactivo con bordes redondeados y efecto cristal (Glassmorphism) que cuenta una historia progresiva con frases tiernas y confesiones en español, francés y portugués.
* **Gran Revelación Final:** Al completarse la conversación, una lluvia de confeti de flores y corazones revela un secreto muy especial de parte de Hello Kitty.

---

## 📂 Estructura del Proyecto
* **`index.html`**: El portal principal de bienvenida. Sirve como menú interactivo para abrir cualquiera de los efectos.
* **`Efectos/`**: Carpeta que contiene todos los archivos individuales de animaciones.
  * **`Espero_te_guste.html`**: La pecera virtual de Hello Kitty (Principal).
  * **`floresamarillas_023.html`**: El clásico detalle de flores amarillas animadas.
  * **`Efecto_Galaxia.html`**: Un universo de estrellas interactivas que siguen tu cursor.
  * **`propuesta_amor_responsiva.html`**: Una propuesta dulce con opción sí/no.
  * *¡Y más de 20 efectos románticos de partículas y corazones!*

## 🚀 ¿Cómo probarlo?

### Localmente:
1. Descarga el repositorio o haz un clon:
   ```bash
   git clone https://github.com/tu-usuario/nombre-del-repo.git
   ```
2. Abre el archivo `index.html` en la raíz con cualquier navegador web (Chrome, Safari, Edge, Firefox). ¡Listo!

### En línea (GitHub Pages):
Este proyecto es 100% estático, lo que significa que puedes subirlo a GitHub y activar **GitHub Pages** para compartir el enlace con esa persona especial de forma gratuita y al instante.
1. Sube este repositorio a tu cuenta de GitHub.
2. Ve a **Settings (Configuración) > Pages**.
3. Selecciona la rama `main` (o `master`) y la carpeta `/ (root)`.
4. ¡Guarda los cambios y comparte el link generado!
